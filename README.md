## طريقة الرفع على GitHub + Vercel

### الخطوة 1 — رفع على GitHub
1. اذهب إلى github.com وسجل دخولك
2. اضغط **New Repository**
3. اسم المستودع: `corevex-erp`
4. اضغط **Create Repository**
5. ارفع جميع ملفات هذا المجلد

### الخطوة 2 — نشر على Vercel
1. اذهب إلى vercel.com
2. اضغط **New Project**
3. اختر مستودع `corevex-erp`
4. اضغط **Deploy**
5. ✅ جاهز! ستحصل على رابط مثل: `https://corevex-erp.vercel.app`

### الخطوة 3 — (اختياري) ربط Firebase للبيانات المشتركة
1. اذهب إلى console.firebase.google.com
2. أنشئ مشروع جديد باسم `corevex-erp`
3. انسخ إعدادات `firebaseConfig`
4. افتح ملف `src/firebase.js` وأضف إعداداتك
